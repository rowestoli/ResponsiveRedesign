Project Overview

This case study presents the redesign of Birds Landing Shooting Sports' website - a local business that offers hunting and clay shooting facilities. The original website presented several usability challenges that affected its efficiency, learnability, and accessibility.

My goal was to create a more easily navigatable, responsive, accessible, and user-friendly design that maintains the current simplicity while significantly improving the user experience across different device types.

Project Scope

  • Usability problem identification
  • Visual design style guide creation
  • Responsive mockups for mobile, tablet, and desktop
  • HTML/CSS implementation
  • Accessibility testing and improvements

Identifying Usability Problems

Efficiency

  • Difficult navigation on desktop - dropdowns not readable
  • Poor information hierarchy - random info on main page
  • No direct path to main services (clay shooting booking)
  • Inconsistent layout between pages

Learnability

  • Confusing menu structure with unclear categories
  • Inconsistent visual cues for clickable elements
  • Non-standard form layout without clear labels

Memorability

  • Inconsistent branding across pages
  • No visual cues to indicate current page
  • Menu items don't follow patterns
  • Different layouts on different screen sizes and pages

WebAIM WAVE Analysis

Using the WebAIM WAVE tool revealed several accessibility issues with the original website:

  • Insufficient color contrast between text and background
  • Missing alternative text for images
  • Lack of proper heading structure
  • Missing form labels for hiring now section
  • No skip navigation links for screen reader users
  • Small touch targets making mobile interaction difficult

These issues significantly impacted users with visual impairments, mobility challenges, and those using assistive technologies.

Conceptual Model Issues

The original website's conceptual model had several disconnects between the user's mental model and the system's implementation:

  • The navigation didn't clearly indicate the site's structure or content hierarchy
  • Critical business information (hours, location) was inconsistently placed and labeled
  • Call-to-action elements didn't stand out visually from regular content
  • Mobile navigation was unintuitive with hidden important functions

Visual Design Process

Style Guide

The style guide established a consistent visual language with a focus on the brand's green and orange colors, clear typography using Dinn Neuzeit Grotesk, and standardized UI components.

Design Decisions

Color & Typography

  • Maintained brand colors (green #076311 and orange #E67E22) for recognition
  • Increased contrast ratios to meet WCAG AA accessibility standards
  • Kept Neuzeit Grotesk font for recognition and style
  • Implemented consistent text sizing and dropdown coloring
  • Decreased white space to improve readability and design

Layout & Navigation

  • Redesigned navigation hovers and button locations for usability
  • Created responsive design adapting seamlessly across devices
  • Impplemented hamburger menu functionality on mobile
  • Prioritized content based on user needs (hours, booking)

Technical Implementation

The redesign was implemented using modern HTML and CSS techniques:

  • Flexbox Layout: Used for responsive navigation and content areas
  • CSS Media Queries: Implemented breakpoints at 480px, 767px, 1024px, and 1920px
  • Semantic HTML: Proper use of headers, sections, and ARIA attributes
  • Custom Font Implementation: Properly loaded with fallbacks

Final Thoughts

This redesign project demonstrates how thoughtful user-centered design can transform a difficult-to-navigate website into an accessible, efficient, and pleasant experience while maintaining brand identity. The implementation of responsive design principles ensures that all users, regardless of their device, can easily access the information they need.
